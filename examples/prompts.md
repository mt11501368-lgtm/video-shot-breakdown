# Example Prompts / 示例提示词

## 完整拉片

```text
帮我完整拆解这个视频。先按叙事镜头校正自动切分，再生成学习分析和模仿复刻两个 Excel。每个镜头都要包含关键帧、准确时间码、画面、运镜、剪辑、声音、台词和创作作用。
```

## 广告学习分析

```text
分析这个广告为什么有效。逐镜解释构图、产品露出、演员动作、情绪曲线、剪辑节奏和声音设计，并总结可以迁移到其他产品的原则。
```

## 短视频复刻

```text
把这个短视频拆成可以直接执行的复刻拍摄表。每行写清楚拍什么、拍多久、演员怎么走、机位和运镜、什么时候剪、说什么、配什么声音，并保持人物和场景连续。
```

## AI 视频提示词

```text
逐镜分析这个 AI 视频，并为每个镜头生成可复用的生成提示词。提示词要包含主体、动作、环境、景别、镜头运动、光线、情绪和跨镜头连续性锚点。
```

## Full Breakdown

```text
Analyze this video completely. Correct the automatic cuts into narrative shots, then generate both learning-analysis and replication Excel workbooks with keyframes, precise timecodes, visuals, camera movement, editing, sound, dialogue, and creative purpose.
```

## Executable Replication

```text
Turn this short video into an executable replication plan. For every shot, specify what to film, duration, actor blocking, camera setup and movement, edit point, dialogue, sound, and continuity anchors.
```
